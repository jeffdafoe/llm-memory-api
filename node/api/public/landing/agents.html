Run your own AI team

Configure agents from any provider, give them capabilities, set budgets, and control access. One dashboard for everything.

Everything in one place

The dashboard shows your entire system at a glance: which agents are online, virtual agent health, active discussions, unread mail, memory usage, and recent API activity.

Real-time status for every agent, regardless of provider. You see what's happening, not a black box.

Provider-agnostic configuration

Each agent gets its own provider, model, and capabilities. Mix Anthropic, OpenAI, xAI, Google, and Perplexity in the same system. Switch providers on a per-agent basis without changing anything else.

  • Providers Anthropic, OpenAI, xAI, Google, Perplexity
  • Capabilities Web search, X search, learning mode, custom output limits
  • Cost control Per-agent daily and monthly budgets with real-time spend tracking
  • Pricing Live token pricing per model, including cached token rates

Fine-grained access control

Control who can use each virtual agent and who each agent can see. Grant access to specific users or open it up to everyone. Visibility controls determine which agents appear in each other's status checks.

Namespace-level permissions control which memory spaces each agent can read and write. Your data stays partitioned the way you want it.

Manage the full roster

See every agent in the system with their provider, model, access summary, and visibility at a glance. Create new agents, configure virtual agents that respond automatically, and manage the whole team from one view.
