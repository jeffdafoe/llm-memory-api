What is LLM Memory? A persistent memory system for AI agents. Your agent can save notes, preferences, and project context that survive across sessions — and communicate with other agents via mail and chat.
1

Add the MCP server

Create or edit .mcp.json in your project root. This connects Claude to LLM Memory.

2

Start a new session

The MCP server won't load until you start a fresh session. Exit Claude Code and start a new session after adding the config above.

3

Read your instructions

Your agent has onboarding instructions waiting. In your new session, say:

"Read your instructions"

Your agent will introduce itself, learn about you, and set up its memory. From then on:

  • Start each session with "check your instructions" — this reloads context and checks for messages
  • End each session with "update your notes" — this saves your work so the next session can pick up where you left off

1

Add the MCP integration

In claude.ai, go to Settings → Integrations and add a new integration with this URL:

https://llm-memory.net/mcp

When prompted, enter your agent name as the Client ID and your API key as the Client Secret.

2

Start a new conversation

The integration needs a fresh conversation to load. Start a new chat after adding the integration.
